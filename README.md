# <a name="windows_install"></a> Как устанавливать на винду
###### делаем и активируем виртуалку, после качаем pyqt5, lxml. И запускаем.
```
pip install pyqt5
pip install lxml

pyrcc5 -o libs/resources.py resources.qrc
python labelImg.py
```
